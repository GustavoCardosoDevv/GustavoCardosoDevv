## 📌 Projetos em destaque

### QuickImóveis
Aplicativo mobile para listagem e visualização de imóveis, desenvolvido com React Native, Expo Router, TypeScript e integração com API REST.

### UniMar
Plataforma de aprendizagem gamificada desenvolvida em parceria com o Labomar, com o objetivo de ampliar o conhecimento sobre a área marinha entre alunos e demais pessoas interessadas.

O projeto busca tornar o aprendizado mais acessível, interativo e organizado, oferecendo conteúdos e recursos voltados à educação ambiental, ciência marinha e conscientização sobre os oceanos, abrangendo principalmente informações sobre o mar de Fortaleza.

### Mercadim
Plataforma de mercado digital desenvolvida com foco na organização e exibição de produtos, navegação intuitiva e experiência simples para o usuário.

O projeto aplica conceitos de desenvolvimento web, componentização, estruturação de dados e criação de uma interface voltada para uso comercial.

### Task.IO - Backend API
API RESTful para gerenciamento de tarefas e listas, desenvolvida com Node.js, Fastify, Prisma, MySQL, JWT, Zod e TypeScript.

### Ecofin
Aplicativo mobile de finanças desenvolvido com React Native, Expo, Firebase e TypeScript.
